# Grad_Thesis

修士論文のLaTeXプロジェクト

## ファイル構成

```
├── .gitignore        # Git除外設定
├── fig/              # 図ファイル用ディレクトリ
│   ├── figX.png      # 図ファイル
│   └── fig.tar.gz    # 図ファイルのアーカイブ
├── latexmkrc         # latexmk設定ファイル
├── main.tex          # 本文（メインファイル）
├── main.pdf          # 出力PDF
├── references.bib    # 参考文献データベース
└── README.md         # このファイル
```
